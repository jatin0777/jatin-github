== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the bootstrap-blog.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Go to Customiser -> Home Page Setting -> Select -> Static Front Page -> Select Any page
5. Create categories, pages and post
6. Select category needed for the homepage
